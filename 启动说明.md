# 🎮 霓虹街机 - 后端启动说明

## 启动后端服务器

### 方法一：使用 npm（推荐）
```bash
npm start
```

### 方法二：直接运行
```bash
node server.js
```

## 启动后

服务器将运行在：**http://localhost:3000**

## 访问网站

在浏览器中打开：
**http://localhost:3000/neon-arcade-login-cn-7.html**

## 测试账号

你可以注册任意账号，例如：
- 用户名：`testuser`
- 邮箱：`test@example.com`
- 密码：`123456`

## 功能说明

1. **登录** - 输入用户名和密码，点击"立即登录"
2. **注册** - 切换到"注册"标签，填写信息后点击"注册账号"
3. 登录成功后会自动跳转到资料完善页面

## 数据库

数据库文件自动创建在：`C:\Users\acer\Desktop\V1.4\database.sqlite`

## 停止服务器

按 `Ctrl+C` 停止服务器
